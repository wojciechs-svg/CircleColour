# CircleColour
Ball Position Marker with embedded shirt and pants colours

interactive and fully customizable, we use HTML, CSS, and a little bit of JavaScript.

This code creates a single "Master Circle" with three color pickers. As you change a color in the menu, the circle updates in real-time.

Key Features of this Code:
Linked Colors: I linked the Outer Frame and the Middle Divider to the same color picker. This ensures that as you change the "grey" part, the thickness remains consistent across the whole frame.

Live Preview: The input event listener in JavaScript means you don't have to click "save"—the circle changes the moment you move your mouse on the color wheel.

Flexbox Layout: It uses display: flex to ensure the top and bottom halves always stay perfectly proportional, no matter what colors you choose.
